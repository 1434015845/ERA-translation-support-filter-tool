# ERA-translation-support-filter-tool
简单的ERA文件过滤工具。
----------------------------------------------------------------
- 为了我自己用AI做批量翻译使用。
- ERB文件中，需要进行翻译的主要为各类PRINT输出的，与玩家直接互动的内容，与;进行标识的注释部分。
- 我使用大佬的 灵瓜 进行翻译时，通过提示词无法达到正则匹配的效果，AI仍然会将IF、SIF、CALL等语句后的日文参数及变量作为文本内容翻译。且因携带大量代码与控制字符，token消耗难以负担。
- 因此该工具将ERB文件分为两部分，以;开头和PRINT开头的，与其他的，并分别输出文件。这样即可实现大幅减少误翻的效果。
- 通过第二个界面，可将翻译后的提取文件与清理文件进行合并。目前我仅做了2000行的单文件测试对比，可以做到完美还原，在更大尺度上不清楚效果如何。

> PS: 建议在进行翻译时使用flash型号模型，经测试pro型号并不会带来显著提升，且仍存在漏翻、错翻、成本高等缺陷。
